# 🏔️ 台灣百岳步程規劃器 (Taiwan Hundred Peaks Hiking Planner)

![System Banner](https://images.unsplash.com/photo-1544198365-f5d60b6d8190?auto=format&fit=crop&q=80&w=1200&h=400)

## 🌟 專案概述
本系統正式名稱為**「上河登山行程規劃器」**，為一個基於 React + TypeScript 開發的現代化網頁應用程式。協助登山者透過選擇國家公園、登山口與每日行程節點，動態計算出支援「上河文化時間標準」的多日縱走預估時間軸。

## 🚀 本地端啟動指南
### 前置作業
- 必須安裝 Node.js 環境。

### 快速啟動三步驟
1. **安裝依賴套件:** 於終端機執行 `npm install`。
2. **設定環境變數:** 在專案根目錄建立 `.env.local` 檔案，並設定 `GEMINI_API_KEY`。
3. **啟動開發伺服器:** 執行 `npm run dev`。
